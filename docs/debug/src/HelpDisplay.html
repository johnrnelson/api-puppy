api-puppy Demo

This is actually the API. Seriously! :-)


This is not another framework or library or even a template, it's just a demo! Get more information about the project at the project web site.

You are going to find bugs because this is just a DEMO! Lots of work needs to be done and things may or may not get worked on. Use Google Chrome and you will get less errors in the DEBU UI. Don't even try to use this on a mobile device.

Check out the GitHub Project to see the source code to see how you can use it in your own projects.

Use the Debug Tool for testing the API to make sure you get what you want before having to test it in your own code.

Have fun with this code. Clone the project and host it yourself. It's easy to learn by breaking things and this is the project to do it in.
