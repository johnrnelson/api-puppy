API Performance and Charts

Working on this but so far the charts are helpful.

Using Charts JS to build a few charts to show whats going on with the API server.

You will need to refresh to see new data. It's not polling or wired into the socket.. yet! :-)

The REST (HTTP/S) and System are seperated just in case the APIServer causes any issues.

Each service has their own stats now. It's a simple map but goes a long way in helping us know our success/failure rate.
